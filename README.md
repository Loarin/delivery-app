\# Delivery Order App



Тестовое веб-приложение для приёмки заказов на доставку.



\## Технологии



\### Backend

\- ASP.NET Core 9

\- Entity Framework Core

\- SQLite

\- FluentValidation



\### Frontend

\- React

\- React Router

\- Axios



\## Функционал



\- Создание заказа с валидацией всех полей

\- Редактирование и удаление уже существующего заказа

\- Просмотр списка всех заказов

\- Детальный просмотр заказа

\- Автоматическая генерация номера заказа

\- Примитивный, но понятный интерфейс
